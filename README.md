# 隐私政策

本应用（方舟生存飞升图鉴助手）非常重视用户隐私。

## 信息收集

本应用不会收集、存储或上传任何个人身份信息。

## 本地数据

应用中的收藏功能仅在用户设备本地存储，不会上传至服务器。

## 网络请求

本应用不包含账号系统，也不会主动向服务器发送用户数据。

## 第三方服务

本应用未接入广告或第三方数据分析服务。

## 权限说明

本应用仅使用系统必要权限（如存储、剪贴板），用于基础功能实现。

## 政策更新

如有更新，本页面将同步更新内容。
